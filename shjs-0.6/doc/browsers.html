SHJS - Browsers

SHJS should work in any browser that conforms to the HTML 4, ECMA-262, and DOM Level 2 standards.

In particular, SHJS has been tested and is known to work with the following browsers:

  • Firefox 2
  • Firefox 3
  • Internet Explorer 6
  • Internet Explorer 7
  • Opera 9.6
  • Safari 3.2
  • Google Chrome 1.0
